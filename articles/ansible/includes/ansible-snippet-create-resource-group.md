1. Cloud Shell で、`rg.yml` という名前のファイルを作成します。

    ```bash
    code rg.yml
    ```

1. 以下のコードをエディターに貼り付けます。

   ```yaml
   ---
   - hosts: localhost
     connection: local
     tasks:
       - name: Create resource group
         azure_rm_resourcegroup:
           name: ansible-rg
           location: eastus
         register: rg
       - debug:
           var: rg
   ```

1. ファイルを保存し、エディターを終了します。

1. `ansible-playbook` コマンドを使用してプレイブックを実行します。

   ```bash
   ansible-playbook rg.yml
   ```
